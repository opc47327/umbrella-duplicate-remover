**监控范围很广**

Umbrella 不只是盯着普通文件夹。它支持：

**判断标准可自定义**

重复文件的认定，Umbrella 支持按文件大小、文件名、修改时间等多种参数组合来判断，减少误判，也可以排除不想监控的子目录。

**处理方式很灵活**

发现重复文件后，你可以：

- 直接扔进系统废纸篓
- 移动到指定备用目录
- 创建硬链接（保留文件可访问性的同时节省空间）
- 调用 Tidy Up 进行更细致的处理

最后这点值得单独说一说：Umbrella 是 Hyperbolic Software 旗下 Tidy Up 的黄金搭档。Tidy Up 负责深度扫描已有的重复文件，Umbrella 负责从今往后预防新增重复。两者配合起来，才是对付重复文件的完整方案。
